Changes to existing tables

EXTEND profiles Add configurable daily start location

EXTEND abwesenheiten Already day-granular — no changes needed

The existing von / bis date range is sufficient. The algorithm treats each date in the range as a fully absent day (zero capacity). Partial-hour absences are out of scope for now.

EXTEND tickets Status 'eingeplant' already exists — no schema changes needed

When a ticket is pulled into a tour, its status is set to 'eingeplant'. When the tour is deleted or the ticket is removed, it reverts to 'offen'.
